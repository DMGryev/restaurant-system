# 🍽️ RestaurantOS — Установка

## Что это?

**RestaurantOS** — современная система управления рестораном с встроенной геймификацией для сотрудников.

## Возможности

✅ Управление заказами и столиками
✅ Меню с категориями и ценами
✅ CRM система с программой лояльности
✅ Геймификация (баллы, достижения, лидерборд)
✅ Аналитика продаж в реальном времени
✅ Авторизация по магнитным картам
✅ Поддержка разных ролей (админ, менеджер, официант, повар)

## Системные требования

- Windows 10/11 (64-bit)
- 4 GB RAM (минимум)
- 500 MB свободного места на диске
- Интернет-соединение

## Установка

1. Запустите **RestaurantOS-Setup-1.0.0.exe**
2. Следуйте инструкциям установщика
3. Выберите папку установки
4. Дождитесь завершения установки
5. Запустите приложение с ярлыка на рабочем столе

## Первый запуск

После установки откроется окно авторизации.

**Тестовые аккаунты:**

| Логин | Пароль | Роль |
|-------|--------|------|
| admin | admin123 | Администратор |
| manager | manager123 | Менеджер |
| waiter1 | waiter123 | Официант |
| cook1 | cook123 | Повар |

## Удаление

Панель управления → Программы и компоненты → RestaurantOS → Удалить
